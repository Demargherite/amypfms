1. AMYFI Smart Financial Management System

Description

Developed a web-based Smart Financial Management System to help users manage personal finances efficiently. The system enables users to track income and expenses, create monthly budgets, monitor financial performance through interactive dashboards, and generate financial reports for better financial planning. Built with a responsive and user-friendly interface to improve accessibility across devices.

Technologies Used

PHP
MySQL
HTML
CSS
JavaScript
Bootstrap

Key Features

Income and expense tracking
Budget planning and management
Financial dashboard and analytics
Monthly financial reports
User authentication
Responsive web interface
