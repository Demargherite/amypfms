AMYFI Smart Financial Management System

Description

Designed and developed a full-stack Smart Financial Management System to help users effectively manage personal finances through digital expense tracking, budget planning, and financial analytics. The system features secure user authentication, real-time financial dashboard visualization, automated report generation, and responsive interfaces to enhance user experience. The project demonstrates strong understanding of software development principles, database design, CRUD operations, and user-centered application development.
